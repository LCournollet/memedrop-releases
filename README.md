# MemeDrop

App qui drop les memes Discord directement sur ton écran.

## Installation

1. Telecharge la derniere version : **[MemeDrop Setup.exe](../../releases/latest)**
2. Double-clic sur le .exe pour installer
3. Si Windows SmartScreen te bloque : clique "Informations complementaires" puis "Executer quand meme"
4. Au premier lancement, clique **User** et entre le code de salon que t'a donne ton admin

L'app tourne ensuite en background. Quand quelqu'un poste un meme dans le channel Discord, ca pop sur ton ecran.

## Comment ca marche

- Clic gauche sur l'icone MemeDrop dans la barre des taches (en bas a droite) = mini popup pour le volume / mute / activer-desactiver
- Clic droit sur l'icone = menu complet
- Bouton "Ouvrir l'app" pour la page de gestion
